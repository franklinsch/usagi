# Views for iOS app

* Projects Overview
  - List of projects
  - People in project
  - Create project (Admin only)

* Project Overview
  - Completion
  - Tasks to complete
  - Completed tasks
  - Participants
  - Settings (Project Manager)

* Tasks 
  - Tasks I need to work on
  - Add Task (Project Manager)

* Create task
  - Task name
  - Estimated time needed
  - Dependant tasks
  - Resources required

* Task view
  - Time spent on task (myself and others)
  - People working on it
  - Resources spent on task
